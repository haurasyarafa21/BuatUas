Langkah-langkah menjalankan aplikasi PhoneBook:

1. Buat virtual environment dari Python3 dan aktifkan.
2. Install requirement dependency dari file req.txt, dengan menjalankan perintah di terminal/command prompt: 
   pip install -r req.txt
3. Jalankan aplikasi dengan mengetikkan perintah di termial/command prompt:
   python manage.py migrate
   python manage.py runserver
4. Cek via browser dengan memasukkan url: localhost:8000,
   Aplikasi berhasil dijalankan jika tampilan pada browser seperti pada image home_page.png
